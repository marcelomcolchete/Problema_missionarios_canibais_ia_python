Siga esses passos para visualizar nosso problema:
1) Abra o terminal
2) Instale o django com o seguinte comando: pip install Django==3.1.2
3) Vá até dentro da pasta onde tem o arquivo "manage.py" cd/projeto
4) Digite no terminal: python manage.py runserver
5) Abra o navegador e acesse http://127.0.0.1:8000/
