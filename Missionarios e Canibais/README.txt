Siga esses passos para visualizar nosso problema:
1) Abra o terminal
2) Instale python3
3) Instale o django com o seguinte comando: pip3 install Django==3.1.2
4) Vá até dentro da pasta onde tem o arquivo "manage.py" cd/projeto
5) Digite no terminal: python manage.py runserver
6) Abra o navegador e acesse http://127.0.0.1:8000/
